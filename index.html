개요가 필요하면 → Codec C-Model Lab (입문/투어). 여기는 심화 과정.

디코더 파이프라인 = 6 심화 앱

각 앱 = HW 파이프라인 스테이지 1개. 비트스트림이 흐르는 순서대로:

회색 카드는 준비 중. 카드를 누르면 해당 tool 심화 페이지로.

이 앱이 다루는 4계층

  1. L1 Spec — decode process 의사코드 · syntax element · 비트 레이아웃 (spec § 인용)
  2. L2 C-Model — AVM 실제 함수/구조체 file:line · 호출그래프 · gdb 실측값
  3. L3 Spec↔Code — 나란히 보기 + AV1→AV2 델타
  4. L4 HW Architecture — datapath · throughput · 라인버퍼 · hazard · 병렬화 (일반 사고법만)

⚠️ 공개 경계

HW 계층은 공개 spec + 오픈소스 코드로 누구나 도출 가능한 일반 사고법까지만 다룹니다. 특정 IP의 라인버퍼 SRAM 정량·대역폭 예산·모듈 추출은 범위 밖입니다.

실측만 · 디코더 한정 · 부동소수 금지(c-model 인용 기준). AVM = BSD, spec = AOMedia.
