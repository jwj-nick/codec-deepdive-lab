이 앱이 다루는 4계층

  1. L1 Spec — decode process 의사코드 · syntax element · 비트 레이아웃 (spec § 인용)
  2. L2 C-Model — AVM 실제 함수/구조체 file:line · 호출그래프 · gdb 실측값
  3. L3 Spec↔Code — 나란히 보기 + AV1→AV2 델타
  4. L4 HW Architecture — datapath · throughput · 라인버퍼 · hazard · 병렬화 · I/O 포맷 · MAC/SRAM 추정

💡 공개 경계

HW 아키텍처 사고 — datapath · I/O 포맷 · per-SB volume · 대역폭 · MAC/SRAM 추정 —는 모두 공개 AVM 소스 + AV2 spec에서 도출되므로 공개합니다. 비공개 선 = 실제 RTL/Verilog 코드 (그것만 private repo).

실측만 · 디코더 한정 · 부동소수 금지(c-model 인용 기준). AVM = BSD, spec = AOMedia.
